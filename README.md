# Лабораторная работа №1: Шифрование текста

## Описание программы
Программа предназначена для шифрования и дешифрования текста на русском языке двумя методами:
- **Столбцовый метод** (Columnar Transposition)
- **Метод Виженера** (Vigenere cipher)

Приложение имеет графический интерфейс, поддерживает работу с файлами и сохраняет неалфавитные символы (знаки препинания, пробелы, цифры) в исходном тексте.

## Установка и запуск
1. Запустите внешний файл `CipherApp.exe`

### Основные операции
- **Шифрование** текста выбранным методом
- **Дешифрование** зашифрованного текста
- **Чтение** исходного текста из файла (.txt)
- **Сохранение** результата в файл
- **Очистка** всех полей ввода
